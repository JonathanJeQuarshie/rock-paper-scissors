# Rock Paper Scissors

A console-based Rock Paper Scissors game built with vanilla JavaScript 
as part of The Odin Project curriculum.

## How to Run

1. Clone the repo
2. Open `index.html` in your browser
3. Open the browser console (F12 → Console tab)
4. Follow the prompts to play against the computer

## How it Works

The player enters their choice via prompt. The computer picks randomly 
using Math.random(). The game runs for 5 rounds and declares a winner.
